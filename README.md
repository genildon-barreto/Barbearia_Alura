# Barbearia Alura

### É um projeto realizado conforme instruído pelo professor Pedro Marins, utilizando conceitos de:

* Estrutura de arquivos HTML
* Estilização de elementos usando CSS
* Utilizando reset.css e posicionamento CSS
* Trabalhando com Design Responsivo
* Utilizando Opacidade e sombra
* Utilizando formulários no HTML
* Importando conteúdos externo: fontes, vídeos e mapas
